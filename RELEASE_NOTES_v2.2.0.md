## OpenFlow Studio v2.2.0 发版说明

### ✨ 核心新特性 (Core Features)
* **多窗口常驻截图与标注：** 新增原生多窗口截图与贴图（Pin）工具，支持画布标注功能。
* **Bitable 数据面板：** 引入基于 React Table 与 Recharts 的 Bitable 数据可视化工作区。
* **模块化设置中心：** 重构并实现了包含系统底层能力的模块化设置中心。
* **OTA 增量更新：** 集成 `electron-updater`，实现全自动检测与后台更新。
* **灵活重命名引擎：**
  * 支持视频与图片多模板独立配置，适配不同业务逻辑。
  * 新增特殊媒体（如创意比特）重命名开关，及独立的序列号计数器。
  * 支持从 JSON 需求单中自动提取制片人信息与独立的项目名解析。
* **游戏词典映射：** 新增游戏字典映射功能，提升自动化处理的准确度。

### 🎨 交互与体验优化 (UI/UX Improvements)
* **拖拽式持久化布局：** 每日工作区支持拖拽重排并持久化保存布局状态。
* **文件夹直接拖拽：** Dropzone 现已支持直接拖拽完整的父文件夹，自动递归扫描内部媒体文件。
* **整理工作区 (Organizer) 增强：** 优化了文件夹生成的 UI 布局、图片预览机制以及快捷打开文件夹的功能。
* **快捷操作与视觉调优：** 优化历史记录展示体验，默认占位图调整，并基于最新的防遮挡 UI 规范重构了 AI 工作区按钮位置。

### ⚡ 性能与架构优化 (Performance & Architecture)
* **并行 I/O 加速：** 采用 `Promise.all` 并发处理项目文件夹初始化、多层级目录创建及尺寸信息读取，大幅缩短 I/O 耗时。
* **目录树缓存：** 引入目录缓存机制，显著降低大批量处理时的文件名碰撞检测与文件处理延迟。
* **主进程依赖打包优化：** 确保 `xlsx` 和 `pinyin-pro` 正确捆绑于主进程，避免客户端打包后抛出模块丢失错误。

### 🐛 稳定性与修复 (Bug Fixes)
* **格式处理器：** 解决部分格式处理转换失败及“清空列表”按钮难以点击的问题。
* **系统级兼容：** 修复 macOS 状态栏托盘图标拉伸问题，以及深色模式下的颜色反转异常。
* **生命周期与崩溃防御：** 彻底修复托盘菜单操作时的“对象已销毁 (Object has been destroyed)”崩溃及启动白屏问题。
* **安全加固：** 对本地持久化存储（Store）增加严格校验，拦截配置存储路径，防止原型链污染 (Prototype Pollution)。
* **容错能力：** 提升文件夹连字符解析能力（支持 `-`、`x` 等多种尺寸分隔符）并优化命名模板短横线过滤。
